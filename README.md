# 🛒 Grocery Store Checkout — Java

A console-based checkout simulator that calculates subtotal, tax, and total for grocery items.

## Description
Simulates a grocery store checkout system. The user inputs quantities of rice (by lb), eggs (by dozen), and bread (by loaf). The program calculates costs, applies a 6.75% tax rate, and prints a formatted receipt.

## Technologies
- Java
- Constants and arithmetic operators
- Scanner for user input

## How to Run
javac GroceryStoreCheckout.java
java GroceryStoreCheckout
